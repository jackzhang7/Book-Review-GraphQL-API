# Graphql

Set-Up
```
git clone https://github.com/jackzhang7/Graphql

cd graphql

node index.js
```

Example Query
```
{
  books {
    id
    title
    author
    genre
    rating
  }
}
```
Visit this link to see a working example: https://graphql-book-api.herokuapp.com/graphiql
